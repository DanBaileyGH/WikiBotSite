Commands

Custom Builds

Custom item builds created by high-ranked players to be as widely applicable as possible for all roles, updated every patch

Commands: ?build [godname], ?b [godname], ?[godname]

God Stats

Get all the stats and details for any God in the game, pulled directly from the Smite API

Commands: ?god [godname], ?g [godname]

God Abilities

View all ability names, descriptions, and statistics for any God

Commands: ?ability [godname], ?a [godname]

Item Stats

Find any item and see its cost, stats, and which other items it builds from or into

Commands: ?item [itemname], ?i [itemname]

God Skins

See all skins for any god, including price, availability, and links to view the card art

Commands: ?skins [godname], ?skin [godname], ?sk [godname]

God Lore

Read the backstory for any God in the game

Commands: ?lore [godname], ?backstory [godname], ?story [godname], ?l [godname]
